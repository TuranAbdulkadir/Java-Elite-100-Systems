# Brainfuck_Interpreter

## Project Overview
An interpreter for the esoteric programming language **Brainfuck**. It executes the 8 minimalist commands (`>`, `<`, `+`, `-`, `.`, `,`, `[`, `]`) on a memory tape, simulating a Turing Machine.

## Tech Stack
* **Memory:** Byte Array (Tape)
* **Logic:** Pointer arithmetic & Loops
