# Flocking_Boids_Simulation

## Project Overview
A simulation of flocking behavior (birds, fish) using Craig Reynolds' **Boids Algorithm**. Independent agents follow three simple rules: Separation, Alignment, and Cohesion to create complex emergent behavior.

## Rules
* **Separation:** Avoid crowding
* **Alignment:** Steer towards average heading
* **Cohesion:** Steer towards average position
