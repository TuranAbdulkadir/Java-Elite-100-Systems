## ðŸš€ How to Run

Since this is a Monorepo containing 100 standalone systems, you can run any project individually.

### Prerequisites
* **Java Development Kit (JDK) 17 or higher**
* **Git**

### Installation
1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/TuranAbdulkadir/Java-Elite-100-Systems.git](https://github.com/TuranAbdulkadir/Java-Elite-100-Systems.git)
    cd Java-Elite-100-Systems
    ```

2.  **Navigate to a project folder (e.g., Blockchain):**
    ```bash
    cd src/Blockchain_Ledger_Core
    ```

3.  **Compile and Run:**
    ```bash
    javac Main.java
    java Main
    ```
